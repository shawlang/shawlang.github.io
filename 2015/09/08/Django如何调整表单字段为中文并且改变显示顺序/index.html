如何调整表单字段为中文并且改变显示顺序

2015-09-08
Django forms
评论

Django中利用forms.form或者ModelForm写表单，默认显示的标签名是英文，也就是字段名。如果想改成英文名怎么办呢？另外表单在模板使用forms.as_ul形式快速生成的，如果想调整字段顺序，怎么办呢？

一、将Django表单字段设置为中文名

一般表单字段是这样设置的：author=forms.CharField(max_length=10)。需要将字段显示为中文名，直接设置label=‘作者’就可以了。这个设置虽然简单，却加深了我对Django forms的理解。

1.这是给表单字段设置初始值，如果没有设置就用默认的值，其实这就是函数传参

2.表单引入forms是这样写的：from Django import forms，from Django.forms import ModelForm。forms首字母是小写，ModelForm，所以forms是一个实例，ModelForm是一个类。forms.CharField()是forms调用CharField()方法；class Meta:是一个内置的类，model=Comment是定义这个类的类属性。从这个类名Meta，也可以知道这个类用来定义类的元数据，有点像配置站点全局信息。很不错的编程思维。

二、调整Django表单字段在模板中显示顺序

在模板中如果使用手动调整字段顺序不是不可以的，如：

<p>{ {forms.author} }</p>
<p>{ {forms.created} }</p>
#author和created表单字段顺序任意调

但是django中可以一次性生成表单，如：

<form>
   { {forms.as_ul} }
</form>
#很多表单的内容都被封装好了。

这时候可以在forms.py中定义表单字段时候使用fields属性调整。

class Meta:
    model=Comment
    fields=['created','author']
#fields是个list，list是有顺序的。dict是无序的。
所以在fields里面确认表单字段的顺序也是有依据的。

三、总结

设置不复杂，但是可以学到Django的一些设计思维。
