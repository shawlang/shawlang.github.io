如何在Django中添加Rss订阅功能

2015-09-14
Django 101
评论

我一般用feedly订阅文章，哎，其实已经资讯过量了。Django中有一个内置的Feed类，可以快速实现Rss订阅工具。

第1步 在views.py引入Feed类

from django.contrib.syndication.views import Feed
from django.core.urlresolvers import reverse
from blog.views import *

class MyFeed(Feed):
    title='所有文章'  
    link='/' 
    description='欢迎订阅，联系邮箱shawlang.com@gmail.com'

    def item(self):
        return Atricle.objects.all() #这里可以自定义
    def item_tile(self,item):
        return item.created
    def item_description(self,item):
        return item.body

    def item_link(self,item):
         return reverse('article',args=[item.pk])

第2步 配置urls.py

from blog.views import MyFeed

urlpatterns=[
  url(r'^feed$',MyFeed(),name='rss')
]

第3步 显示在模板上

<p><a href="{ % url rss % }" />Rsss订阅</a></p>

总体来说，Rss订阅器写法固定，也许是自己还没有深入理解。
