Django全局变量的设置

2015-08-21
Django>Django全局信息设置
评论

CMS中都有个站点信息配置，并且配置一次整个网站都生效。这个信息配置就是站点全局变量配置。Django中一般都是通过上下文处理函数（context-proccessor）来处理。

第一步：在settings.py配置全局变量

`SITE_NAME`='个人网站'
'SITE_DESC'='这是一个很好玩的个人网站'
'SITE_TITLE'='首页'

第二步：定义上下文处理函数（context processors）

views.py

from django.conf import settings
from django.shortcuts import render,render_to_response
from django.template import RequestContext

def siteinfo(request):
    return {"SITE_NAME":settings.SITE_NAME,"SITE_DESC":settings.SITE_DESC,
    "SITE_TITLE":settings.SITE_TITLE}

渲染模板的两种方法：

def index(request):
    return render(request,"index.html",locals())

def index(request):
    return render_to_response('index.html',{'message':'hello'},context_instance=RequestContext(Request,processors=[info]))

#这两种方法的原理是把siteinfo函数的context加到index函数渲染函数的context里
#所以如果使用render渲染模板的话，须返回locals()，如果使用render_to_response渲染，须借助RequestContext类库

第三步：将上下文处理函数添加到settings.py中

#如下：将siteinfo处理函数添加到context_processors的字典中
#格式是：siteinfo的路径

TEMPLATES = [
{
    'BACKEND': 'django.template.backends.django.DjangoTemplates',
    'DIRS': [os.path.join(BASE_DIR,'templates')],
    'APP_DIRS': True,
    'OPTIONS': {
        'context_processors': [
            'django.template.context_processors.debug',
            'django.template.context_processors.request',
            'django.contrib.auth.context_processors.auth',
            'django.contrib.messages.context_processors.messages',
            'blog.views.siteinfo'
        ],
    },
  },
]

RequestContext详解

1.模板渲染的3种方法

#第一种
from django.http import HttpResponse
from django.template import loader,Context
def index(request):
    t=loader.get_template('index.html')
    c=Context({})
    return HttpResponse(t.render(c))

#第二种
from django.shortcuts import render
from django.conf import settings

def index(request):
    site_name=settings.SITE_NAME
    return render(request,'index.html',locals)

#第三种
from django.shortcuts import render_to_response

def index(request):
    return render_to_response('index.html',context)

2.RequestContext是Context类的一种

RequestContext实例化对象的时候需要3个参数：request','context','processors
第3个参数processors是可选的，如果不传RequestContext与Context作用相同.并且processors是一个list。这个list是和settings.py中context_processors的list。

#第一种方法：使用t.render(c)，注意要导入RequestContext
from django.http import HttpResponse
from django.template import loader,RequestContext

def index(request):
    t=loader.get_template('index.html')
    c=RequestContext(request,context,processors=[custom_procesors])
    return HttpResponse(t.render(c))

#第2种方法

from django.shortcuts import render

def index(request):
    return render(request,'index.html',locals())

#第3种方法：使用render_to_response

from django.shortcuts import render_to_response
from django.template import RequestContext

def index(request):
    render_to_response('index.html',context,context_instance=RequestContext(request,processors=[custom_processors]))
