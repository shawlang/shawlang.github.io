使用内置的表单类forms生成表单

2015-08-30
Django forms
评论

Django很重，是因为封装了很多常用的东西。网站中的表单是一个很常用的功能组件，如注册登录框、联系人、调查表等等都需要表单。Django可以用forms.py定义表单字段，然后可以很方便的在前台调用显示出来。

步骤一：在myapp下建立forms.py文件，开始定义表单

在forms.py里定义表单非常像在models.py定义数据库表和字段。

#forms.py

from django import forms

#forms类非常智能，不仅可以直接定义好标签，还可定义好属性
class EnrollForm(forms.Form):
    name=forms.CharField(max_length=20)
    phone_num=forms.CharField(max_length=11)
    email=forms.EmailFeild(required=False)
    message=forms.CharField(widget=forms.Textarea)

步骤二：在views.py处理表单请求

Django默认是开启csrf这个中间件的，所以在处理请求的时候也需要进行csrf验证。

#views.py
#需要导入csrf上下文处理器csrf类
from django.template.context_processors import csrf
from myapp.forms import EnrollForm
from django.shortcuts import render_to_response

def enroll(request):

    if request.method=='POST':
        form=EnrollForm()
        if form.is_valid():
        #一旦表单通过验证，就会自动生成一个字典：forms.cleaned_data
           name=form.cleaned_data['name']
           phone_num=form.cleaned_data['phone_num']
           email=form.cleaned_data['email']
           message=form.cleaned_data['message']
    form=EnrollForm()
    context={}
    #csrf是用来防止伪装提交请求验证
    context.update(csrf(request))
    context['form']=form
    return render_to_response('enroll.html',context)

步骤三：在模板中显示form
