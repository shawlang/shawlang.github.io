什么是args和kwargs

2015-08-17
Python>Python语法
评论

Python中形式参数args和kwargs是什么呢？有时让我有点不解。看到这篇帖子之后，彻底明白了。

args是一个tuple，*kwargs是一个dictionary。

def test(*args,**kwargs):
    print 'args is:',args
    print 'kwargs is:',kwargs

test(1,2,3,4)
output: 
    args is:(1,2,3,4)
    kwargs is:{}

test(1)
output:
    args is:(1,)
    kwargs is:{}

test(a=1,b=2,c=3)
output:
    args:()
    kwargs:{'a':1,'b':2,'c':3}

test(6,a=1,b=2,c=3)
output:
    args:(6,)
    kwargs:{'a':1,'b':2,'c':3}

test(5,6,a=1,b=2,c=3)
output:
    args:(5,6)
    kwargs:{'a':1,'b':2,'c':3}

test((1,2,3),{'a':1,'b':2,'c':3})
output:
    args:((1,2,3),{'a':1,'b':2,'c':3})
    kwargs:{}

但是要注意args始终要放在**kwargs前面，否则会报错：SyntaxError: non-keyword arg after keyword arg。这个args和**kwargs写嵌套函数或者封装函数特别有用。
