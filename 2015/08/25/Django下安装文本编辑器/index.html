Django下安装文本编辑器

2015-08-25
Django>插件工具
评论

任何页面都可以安装一个文本编辑器的插件，原理主要是在页面引入相关的JS、Css文件，并根据文本编辑器的API实例化这个插件。Django中引入文本编辑器插件，主要是注意这个JS、Css文件怎么引入？这就需要Django中Media这个类了。

一、在Django admin管理后台添加文本编辑器

Django admin后台的样式是使用admib.ModelAdmin这个类定义的。添加文本编辑器也在这个类里面添加。

#admin.py
class ArticleAdmin(admin.ModelAdmin):
    list_display=('title','content','pub_time')

    class Media:
        #添加JS文件 注意这是个Tuple，如果js文件是单个文件，后面一定要有个逗号
        js=('/static/js/editor/editor.js',\
            '/static/js/editor/config.js'
        )

        #添加css文件,这与js有点不同，使用dict添加。
        css={'all':('/static/css/editor.css',)}

#config.js 一般需要自定义一个JS配置文件
#因为Django自动把JS文件加载在head标签里，所以使用window.onload这个事件初始化文本编辑器
window.onload=function(){
    var editor=new Editor()
}
