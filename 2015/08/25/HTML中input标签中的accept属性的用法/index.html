HTML中input标签中的accept属性的用法

2015-08-25
HTML>Html语法
评论

HTML中input标签中有个accept属性，一般这个属性是在需要进行文件上传的时候使用，这些文件包括视频、音频和图片，还有一些自定义约定的文件类型。

作用：显示的时候只显示指定的文件类型的文件

一. 图片类型

1.接受所有的文件类型

<input type="file"  multiple="multiple" accept="image/*" />

2.接受指定的图片类型

<input type="file" multiple="multiple" accept="image/gif,image/jpeg" />

二. 视频类型

<input type="file" mulitple="multiple" accept="video/*" />

三. 音频类型

<input type="file" multiple="multiple" accept="audio/*" />

四. 混合类型

将各种类型混合的时候，只需要在accept中用逗号隔开就行。

<input type="file" multiple="multiple" accept="image/*,audio/*" />
