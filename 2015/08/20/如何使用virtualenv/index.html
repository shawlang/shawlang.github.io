如何使用virtualenv

2015-08-20
Python>Python工具
评论

virtualenv是python三大神器之一，用于给计算机上各个应用一个独立的运行环境，互不干扰，一个计算机既可以运行python2.7版本的应用，也可以运行python3.4版本的应用。

1. 安装virtualenv

一般情况下，使用命令pip install virtualenv即可，如果出现错误can't fetch the index base url可以看这篇文章。

pip install virtualenv

2. 安装虚拟环境

创建一个自己准备开展项目的文件夹，如myproject。

cd F:/myproject
#安装env 注意这个`env`只是一个名字而已，可以自定义
virtualenv env

3. 激活虚拟环境

安装好虚拟环境以后，在myproject目录下有个新目录env。

cd F:/myproject/env
cd F:/myproject/env/scripts
activate

激活以后控制台的路径前面会有一个标志<env>。此时，可以使用pip安装自己需要的软件了

cd F:/myproject
pip install django

4. 退出虚拟环境

cd F:/myproject/env/scripts/deactivate

5. windows下注意事项

在Windows下路径会与*nix下略有不同:脚本和可执行文件在Windows下位于ENV\Scripts\下，而非ENV/bin/，模块也会安装在ENV\Lib\下，而非ENV/lib/。要在某个含有空格的目录下面创建virtualenv环境，就要安装win32api。

可以参考中文文档：http://virtualenv-chinese-docs.readthedocs.org/en/latest/#windows
