windows上安装virtualenv出现错误：Cannot fetch index base URL

2015-08-20
Python>Python工具
评论

在windows上使用pip安装Virtualenv出现错误：Cannot fetch index base URL:https://pypi.python.org/simple/，然后翻个墙，安装成功了。那么不翻墙，怎么办？

Google到了这篇文章，python安装包管理工具pip以及修改pip源，这篇文章说得很详细。本文重写一下，做为一个笔记。

在python 2.7中pip工具一般都是默认安装的，也可以手动安装。官网文档在这里：https://pip.pypa.io/en/stable/installing.html。

  1. 到这里来下载get-pip.py
  2. 在控制台python get-pip.py
  3. 还是会报错:cannot fetch index base url…
  4. 还是先翻墙吧

在此记录一下，pip版本升级的命令：pip install -U pip

问题解决办法：

国内访问不到官方pip源，可以访问国内镜像pip源。

1. 直接指定镜像地址
pip install virtualenv -i https://pypi.tuna.tsinghua.edu.cn/simple
2. 设置为默认

修改pip的配置文件，如果没有配置文件，就创建一个。Linux创建为pip.conf，windows下创建pip.ini。

[global]
index-url=https://pypi.tuna.tsinghua.edu.cn/simple
3. 镜像列表

. 官方: https://pypi.python.org/pypi/simple/

. 清华大学：https://pypi.tuna.tsinghua.edu.cn/simple。注意这个url后面的simple不能少，而且https那个s不能少。

. 豆瓣: http://pypi.douban.com/
