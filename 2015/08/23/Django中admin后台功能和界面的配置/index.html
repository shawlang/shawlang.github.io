Django站点后台管理

2015-08-23
Django>Django admin
评论

Django站点后台管理使用的admin这个类库。总的来说，使用admin来配置后台，分为两个部分，一个是实现后台应有的功能，另一个是实现界面样式。

一. 使用Django admin连接数据库

  1. 确认django.contrib.admin在settings.py中的INSTALLED_APPS中。

  2. 确认urls.py已配置好后台的路由：url(r'^admin/', include(admin.site.urls))

  3. 在浏览器中打开localhost:8000/admin测试一下。

4.测试成功后，打开admin.py，进行如下配置

from django.contrib import admin
from models import *

admin.site.register(Article)
  1. 再次同步一下数据库(先python manage.py makemigrations blog，然后python manage.py migrate blog)，打开浏览器登录后台测试。

二. 使用类库修改admin后台管理的样式

因为Django是比较成熟的框架，admin都是自动生成的。定义一些样式，不是用css，js，反而是定义一些类。

#admin.py

from django.contrib import admin
from models import *

class ArticleAdmin(admin.ModelAdmin):
    list_display('title','content','pub_time')
#这里定义一个专门用来管理后Article这个数据表的样式
#list_display决定并列显示多少个字段

admin.site.register(Article,ArticleAdmin)
#注册定义样式的类
