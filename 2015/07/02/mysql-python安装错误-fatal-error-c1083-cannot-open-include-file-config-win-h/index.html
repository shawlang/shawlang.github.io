mysql-python安装错误 fatal error c1083:cannot open include file:config-win.h

2015-07-02
Django>入门学习笔记
评论

最新版的django 1.8.1连接mysql有点问题，先要安装VCForPython27.msi，安装完成了以后，需要安装mysql-python，但是pip install mysql-python总是不成功，报这样的错：

fatal error C1083: Cannot open include file: ‘config-win.h’: No such file or directory

然后Google一下，发现网上也有好多童鞋遇到这样的问题。解决问题的方法就是找一个对应python需要的msql-python包。先要在DOS下python一下MSC位数，如果是32位的就下载32位的mysql-python，如果是64位的就下载64位的mysql-python。下载地址为：https://pypi.python.org/pypi/MySQL-python/或者在github下载

另一个配置django的数据库的时候，注意端口号（port）是3306，不能以为是8000。
