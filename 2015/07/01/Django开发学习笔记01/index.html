Django开发学习笔记（1）

2015-07-01
Django>入门学习笔记
评论


原来系统地学习过django，但只是看视频，没有做详细笔记，没有做总结。现在发现学一样东西，做一个总结，写一篇博客，学得更加扎实，即使以后有所忘记，也可以根据博客很快的熟悉起来，而不用从头开始学。此为django学习的第一篇。

1.开发环境

如何在Windows7 Dos知道python和django的版本呢？在Windowns7 DOS下输入如下命令就可以得到。

>>python
python 2.7.9
>>import django
>>django.get_version()
'1.8.1'

2.创建第一个Project和APP

cd D:\  
#进入D盘

django-admin startproject mediacms 
#使用命令 django-admin startproject 创建项目 mediacms

cd mdediacms   
#进入项目 mediacms 文件夹

django-admin startapp blog  
#使用命令 django-admin startapp 创建APP blog

3.让自己创建的APP生效

因为 project mediacms 可以创建很多APP，所以创建后需要手动启用这些APP。在 project mediacms 中 setting 文件中找到元组INSTALLED_APPS，在末尾行添加 ‘blog’即可。

4.启用django内置的服务器，查看效果

python manage.py runserver
在浏览器中输入 localhost:8000 或者 127.0.0.1:8000:会有如下提示

It worked!
Congratulations on your first Django-powered page.

5.总结

常用命令：django-admin startproject,django-admin startapp,python manage.py runserver
