当在浏览器输入一个网址，发生了什么？一般来说，网址先转换为IP地址，然后通过HTTP协议发送给服务器。那么在服务器端，怎么走Django框架里的一套流程呢？

1.在views.py中建立一个视图函数

因为涉及到HTTP协议，所以需要导入HttpResponse类。

from http import HttpResponse
def hellodjango(request):
    return HttpResponse("hello I'm django")

2.在urls.py中配置URL

from blog.views import * #把views.py中写的视图函数导入到urls.py中
url(r'',hellodjango)
#这里url为空，这样就在浏览器中输入`localhost:8000`，就可以直接执行hellodjango视图函数。

3.访问

在windows dos条件下，启动服务器：`python manage.py runserver`

4.总结

现在还没有涉及到MODEL和模板，不过当输入网址后的大致流程如下：

  • Django
  • python
  • 创建视图函数
