Javascript DOM事件详解

2015-07-25
Javascript>DOM事件
评论


Javascript中DOM的全称是Document Object Model，即文档对象模型。DOM是一种将XML或HTML文档解析成树形节点的方法。DOM事件是文档、文档元素以及与用户交互产生的一系列的行为。当这些行为发生的时候，如果我们对这些行为做出相应的处理就会实现我们的一些实际开发中的需求，例如一些特效。DOM事件处理主要包含这3个方面：事件流，事件处理程序以及事件对象。
